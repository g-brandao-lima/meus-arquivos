O que é

  • CLAUDE.md = instruções permanentes para o Claude

Contexto

  • Quem você é (PM de fintech, SaaS...)

  • Tom (direto, analítico, criativo)

  • Metodologia (SDD)
  • Regras de validação

SDD EM AÇÃO

Feature: "Notificação de abandono de carrinho"

PM escreve

Spec

  • Reduzir abandono em 15%
  • Push 30min após abandono
  • Max 1 notificação por carrinho
  • Não enviar se já comprou
PM define

Cenários de Teste

  • Happy: abandona, recebe em 30min
  • Edge: compra em 20min = não recebe
  • Edge: 2 abandonos = 1 push
  • Erro: push fora = retry 5min
Claude executa

Gera

  • PRD completo
  • User stories INVEST
  • Critérios de aceitação técnicos
PM valida

Revisão

  • Confere contra cenários
  • "A IA cobriu o edge case X?"
  • Ajusta e aprova

PESQUISA DE MERCADO

Como o Claude mapeia seu mercado em minutos

PESQUISA COM CLAUDE

Nível 01 - Rápido

Claude (chat)

  • Deep thinking para análise
  • Mapeia temas e concorrentes
  • Gera hipóteses iniciais
  • Gratuito ou $20/mês

Nível 02 - Profundo

Claude + MCPs

  • Busca em tempo real (Brave Search)
  • Navega sites concorrentes (Playwright)
  • Processa dados locais (Desktop Commander)
  • Raciocínio estruturado (Sequential Thinking)

MCPs ESSENCIAIS

Model Context Protocol: Claude usa ferramentas profissionais

Playwright

Claude controla navegador automaticamente. Testa fluxos, extrai dados de concorrentes.

Sequential Thinking

Claude 'pensa em voz alta' antes de responder. Raciocínio completo e validado.

Brave Search

Claude pesquisa a internet em tempo real. Dados atualizados, reviews, tendências.

Desktop Commander

Claude acessa seus arquivos. Processa planilhas, executa scripts, analisa dados.

CASO REAL: MERCADO LIVRE

Problema

Concorrência intensa entre ferramentas para perfil de vendedores.

Solução

  • Claude mapeou ecossistema inicial
  • MCP aprofundou: fóruns, reviews, docs
  • Playwright testou interfaces automaticamente

Resultado

  • Descoberta de gaps que ninguém atendia
  • Padrões de reclamação recorrentes identificados

ENTREVISTAS SINTÉTICAS

Simule usuários e ganhe hipóteses em minutos

PERSONAS SINTÉTICAS COM CLAUDE

Por que funcionam

  • Disponíveis 24/7
  • Exploram cenários extremos
  • Geram mais hipóteses em menos tempo
  • Sem viés de conveniência

Construção em 5 passos

  1. Desconstrua o produto
  2. Triangulação de contexto
  3. Gere arquétipos hipotéticos
  4. Crie prompt estruturado
  5. Entreviste a persona

Claude Cowork: contexto compartilhado com o time em tempo real

FRAMEWORK DE BOAS PERGUNTAS

Fazer

  • Aprofundar no porquê:
    "Me conta mais sobre isso.."
  • Usar os 5 tipos:
    "O quê? Por quê? Como? Quando? Onde?"
  • Basear em fatos:
    "Qual foi a última vez que.."

Evitar

  • Resposta embutida:
    "Você concorda que.."
  • Começar fechado:
    Sim/não sem follow-up
  • Enviesar com contexto demais:
    Influenciar a resposta
